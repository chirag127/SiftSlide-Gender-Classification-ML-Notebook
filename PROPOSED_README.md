## üìà Project Overview

`SiftSlide` is an archived project that served as a deep dive into applying machine learning for binary classification tasks, specifically focusing on gender classification. The repository contains a comprehensive Jupyter Notebook (`gender_classification_exploration.ipynb`) that walks through the entire machine learning lifecycle: data loading, exploratory data analysis (EDA), feature engineering, model selection, training, evaluation, and interpretation.

This project was instrumental in understanding the practical challenges and methodologies involved in building and assessing classification models within the data science domain.

## ‚ú® Key Features

*   **Comprehensive ML Workflow:** Covers data preprocessing, feature engineering, model training, and evaluation.
*   **Exploratory Data Analysis (EDA):** Detailed analysis and visualization of the dataset to uncover insights.
*   **Model Experimentation:** Includes implementations and comparisons of various classification algorithms (e.g., Logistic Regression, SVM, Random Forest).
*   **Performance Metrics:** Utilizes key metrics like Accuracy, Precision, Recall, F1-Score, and ROC AUC for thorough model assessment.
*   **Jupyter Notebook Environment:** A well-commented and structured notebook for interactive learning and reproducibility.
